# journalism2026

Рейтинг закладів вищої освіти України за середнім конкурсним балом
вступників на спеціальність **061 «Журналістика»** (бакалавр і магістр).

Рейтинг рахується за даними ЄДЕБО й показує середній конкурсний бал та
кількість заяв по кожному закладу освіти окремо для бакалаврату й
магістратури.

🔗 Локально: відкрийте `index.html` подвійним кліком, або
```
python -m http.server 8080
```
і зайдіть на http://localhost:8080

## Статус: демо-дані

**Зараз сайт показує згенеровані тестові дані** (`js/data.js`), а не реальний
фід ЄДЕБО. Список закладів освіти — справжні українські університети, що
готують журналістів, але бали й кількість заяв для кожного з них — випадкові,
детерміновано згенеровані для демонстрації дизайну та інтерфейсу (перемикач
бакалавр/магістр, слайдер по днях, рейтингова таблиця).

Причина: сесія, в якій верстався сайт, не мала мережевого доступу до
`vstup.edbo.gov.ua`, тож підключити й перевірити реальний фід звідти було
неможливо.

## Що всередині

| Файл | Призначення |
|---|---|
| `index.html` | Єдина сторінка дашборду |
| `css/style.css` | Стилі: редакційний дизайн (Fraunces + Manrope), хіро-блок, стата-плитки, п'єдестал топ-3, картки |
| `js/data.js` | Дані: список ЗВО (реальний) + генератор тестових щоденних знімків |
| `js/app.js` | Рендер п'єдесталу й таблиці, перемикання бакалавр/магістр, стрічка дат, дельти рангу за добу, розгортання списку |

## Підключення реальних даних ЄДЕБО

Щоб дашборд показував справжні цифри, потрібно:

1. **Отримати доступ до даних vstup.edbo.gov.ua.** Публічно на сайті доступні
   рейтингові списки вступників по кожній конкурсній пропозиції (заклад освіти
   + спеціальність + рівень). Потрібно з'ясувати точний контракт: чи це
   відкритий API (SPA vstup.edbo.gov.ua кудись звертається за даними — дивіться
   вкладку Network у DevTools браузера під час перегляду рейтингового списку),
   чи треба парсити HTML/експортовані XLSX-файли по кожній пропозиції окремо.
2. **Написати скрипт агрегації**, який:
   - знаходить усі конкурсні пропозиції зі спеціальністю 061 «Журналістика»
     (окремо бакалавр і магістр) для всіх ЗВО;
   - для кожної пропозиції рахує середній конкурсний бал і кількість заяв;
   - зберігає щоденний знімок (наприклад, `data/2026-07-21.json`) у форматі,
     який очікує `js/app.js` (див. структуру `DB.snapshots` у `js/data.js`).
3. **Автоматизувати через GitHub Actions** (`.github/workflows/update-data.yml`,
   заготовку додано, зараз запускається лише вручну — `workflow_dispatch`).
   GitHub Actions виконується на серверах GitHub з повним доступом в інтернет,
   тож не залежить від мережевої політики агента. Після підтвердження точного
   джерела даних — розкоментуйте `schedule:` у workflow, щоб оновлювати дані
   щодня автоматично, і замініть генератор у `js/data.js` на `fetch("data/...")`
   реальних знімків.

## Формат знімка даних

```js
{
  date: "2026-07-21",
  asOf: "2026-07-21T23:30:00+03:00",
  bachelor: [
    { rank: 1, id: "knu", name: "…", short: "КНУ", hue: 350, score: 178.1, applications: 646 },
    …
  ],
  master: [ … ],
  totalApplications: { bachelor: 3262, master: 1265 }
}
```

## Ліцензія на дані

ЄДЕБО — державна база даних; рейтингові списки вступників публікуються
відкрито в межах вступної кампанії відповідно до законодавства про вступ до
закладів вищої освіти України. Перед публічним запуском варто звірити умови
використання цих даних (vstup.edbo.gov.ua) з юристом.
